## Название проекта

Лабораторная работа № 6, задание 1. Приложение для управления фоном на языке Swift.

## Описание

Эта ветка содержит каталог `task1` с исходным кодом iOS-приложения, которое
меняет фон экрана с помощью `UISwitch`.

В положении `ON` используется изображение `bg1.jpg`, в положении `OFF` —
изображение `bg2.jpg`.

## Установка

1. Откройте Xcode на macOS.
2. Создайте iOS-проект типа `App` на языке `Swift` с использованием `Storyboard`.
3. Добавьте в проект файлы из каталога `task1`.
4. Подключите изображения `bg1` и `bg2` в `Assets.xcassets`.
5. Свяжите элементы интерфейса со свойствами и действием контроллера.

## Использование

1. Запустите проект в симуляторе iPhone.
2. Переключайте `UISwitch`.
3. Проверьте изменение фона и текста индикатора.
