## Название проекта

Лабораторная работа № 6, задание 4. Мобильное приложение с использованием SwiftUI.

## Описание

Эта ветка содержит каталог `task4` с приложением на `SwiftUI`, созданным по
мотивам примера `SwiftUI: Getting Started`.

В приложении используются декларативные представления, навигация между
экранами и реактивное обновление интерфейса.

## Установка

1. Откройте Xcode на macOS.
2. Создайте iOS-проект на языке `Swift` с поддержкой `SwiftUI`.
3. Добавьте файлы из каталога `task4`.
4. Проверьте целевую платформу и сборочную схему.

## Использование

1. Запустите приложение в симуляторе.
2. Проверьте главный экран, работу элементов SwiftUI и переход между сценами.
3. Убедитесь, что данные и состояние интерфейса обновляются корректно.
