## Название проекта

Лабораторная работа № 6, задание 5. Двухэкранное мобильное приложение на Swift.

## Описание

Эта ветка содержит каталог `task5` с приложением на `Swift` для расчёта скидки
по варианту 17.

Поддерживаются два режима:

- скидка по количеству товаров;
- скидка по стоимости покупки.

Выбор режима выполняется через `segmented control`. Приложение рассчитано на
две сцены и поддерживает локализацию.

## Установка

1. Откройте Xcode на macOS.
2. Создайте iOS-проект на языке `Swift` с использованием `SwiftUI` или `UIKit`.
3. Добавьте файлы из каталога `task5`.
4. Подключите локализации `ru`, `en`, `be`.
5. Проверьте сборку под разные размеры экранов и ориентации.

## Использование

1. Запустите приложение в симуляторе iPhone.
2. Выберите тип скидки через `segmented control`.
3. Введите количество товаров или стоимость покупки.
4. Перейдите на экран результата и проверьте рассчитанную скидку.
