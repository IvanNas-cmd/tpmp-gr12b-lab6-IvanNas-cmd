## Название проекта

Лабораторная работа № 6. Разработка простых мобильных приложений на языках
Swift и Objective-C.

## Описание

Этот репозиторий содержит лабораторную работу № 6 по дисциплине
`Технологии программирования для мобильных приложений`.

Автор: `Насенник Иван`, группа `12б`.  
Вариант: `17`.

В репозитории реализованы задания 1–5:

- `task1` — приложение для управления фоном на Swift;
- `task2` — приложение погоды и сведений о городе на Objective-C,
  вариант 17 с музеями;
- `task3` — приложение для расчёта BMR, BMI и суточной нормы калорий на Swift;
- `task4` — мобильное приложение на SwiftUI по мотивам примера
  `SwiftUI: Getting Started`;
- `task5` — двухэкранное приложение на Swift для расчёта скидки,
  вариант 17.

## Установка

Для использования исходников нужны:

- macOS;
- Xcode 26.x или новее;
- iOS SDK;
- подключённые графические ресурсы и локализации, указанные в README каждого
  задания.

## Использование

Структура веток репозитория:

- `main` — README, отчёт и итоговая объединённая версия всех заданий;
- `feature-task1` — задание 1;
- `feature-task2` — задание 2;
- `feature-task3` — задание 3;
- `feature-task4` — задание 4;
- `feature-task5` — задание 5.

Задание 2.2 для варианта 17:

- при выборе города показываются температура, изображение известного музея и
  название музея;
- используются города из Грузии и Франции.

Задание 5 для варианта 17:

- расчёт скидки по количеству товаров;
- расчёт скидки по стоимости покупки;
- выбор типа скидки через `segmented control`;
- локализация на русский, английский и белорусский языки.
