## Название проекта

Лабораторная работа № 6, задание 3. Приложение для расчёта калорий и индекса массы тела на Swift.

## Описание

Эта ветка содержит каталог `task3` с iOS-приложением на `Swift`, которое
рассчитывает индекс массы тела `BMI`, базовый обмен веществ `BMR` и суточную
норму калорий с учётом пола и уровня активности.

В проекте используются `TextField`, `SegmentedControl`, `Label` и кнопка запуска
вычислений.

## Установка

1. Откройте Xcode на macOS.
2. Создайте iOS-проект типа `App` на языке `Swift`.
3. Добавьте файлы из каталога `task3`.
4. Подключите элементы интерфейса из `Storyboard`.
5. Проверьте связи `IBOutlet` и `IBAction`.

## Использование

1. Запустите приложение в симуляторе.
2. Введите возраст, рост и вес.
3. Выберите пол и уровень активности.
4. Нажмите кнопку расчёта и проверьте результат.
