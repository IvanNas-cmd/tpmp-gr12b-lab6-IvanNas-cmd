## Название проекта

Лабораторная работа № 6, задание 2. Приложение погоды и сведений о городе на Objective-C.

## Описание

Эта ветка содержит каталог `task2` с iOS-приложением на `Objective-C`.
Приложение показывает температуру выбранного города, подсвечивает её цветом и
дополнительно выводит изображение известного музея и его название.

Для варианта 17 использованы города из Грузии и Франции, не менее двух городов
из каждой страны.

## Установка

1. Откройте Xcode на macOS.
2. Создайте iOS-проект типа `App` на языке `Objective-C` с использованием `Storyboard`.
3. Добавьте файлы из каталога `task2`.
4. Подключите локализации `ru`, `en`, `be`.
5. Добавьте изображения музеев в `Assets.xcassets`.
6. Свяжите элементы интерфейса с `IBOutlet` и `IBAction`.

## Использование

1. Запустите приложение в симуляторе iPhone.
2. Выберите город.
3. Проверьте отображение температуры, цвета температуры, названия музея и изображения.
