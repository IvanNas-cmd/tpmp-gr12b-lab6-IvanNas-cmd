# Лабораторна робота №6: Розробка простих мобільних додатків на мові Swift та Objective-C

**Автор:** Насеннік Іван, Група 12б  
**Варіант:** 17

## Опис

Лабораторна робота з дисципліни «Технології програмування для мобільних додатків».  
Реалізовано 5 завдань на мовах Swift та Objective-C для платформи iOS.

## Структура гілок

| Гілка | Опис |
|---|---|
| `main` | README з описом проєкту та посиланнями |
| `feature-task1` | Завдання 1 — Перемикач фону (Swift + UIKit) |
| `feature-task2` | Завдання 2 — Погода та музеї міст, варіант 17 (Objective-C) |
| `feature-task3` | Завдання 3 — Калькулятор BMR/BMI (Swift + UIKit) |
| `feature-task4` | Завдання 4 — Базовий SwiftUI додаток |
| `feature-task5` | Завдання 5 — Калькулятор знижок, варіант 17 (Swift + SwiftUI) |

## Варіант 17

**Завдання 2.2:** При виборі міста показує зображення відомого музею, що знаходиться в місті, та назву музею.  
Міста: 2 з Грузії, 2 з Франції.

**Завдання 5:** Розробити застосунок-калькулятор знижки за покупку залежно від кількості товарів чи вартості покупки.  
Знижки: по кількості (1% — 10 товарів, 3% — 20 товарів) і вартості (3% — >50 грн, 5% — >100 грн).  
Поле segmented control для вибору типу знижки. Локалізація: RU, EN, BE.
